Web Map Service

The primary responsibility of a web map service (WMS) is to produce georeferenced images from geographic information. The Open Geospatial Consortium (OGC) first released the WMS standard, version 1.0.0 on April 19, 2000. Version 1.3.0 is the current standard which defines three operations GetCapabilities, GetMap, and GetFeatureInfo following the OGC Web Services Context (OWS).

Web Feature Service

The web feature service (WFS) is an International Standard which defines eleven operations. This standard allows clients the ability to create, modify, and exchange geographical information via the Internet. The feature data is exchanged utilizing Geography Markup Language (GML).
